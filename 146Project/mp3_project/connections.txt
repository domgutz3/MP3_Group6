SJtwo to MP3 Decoder:
P0.1 (Top Left SCK2) -> SCLK
P1.1 (MOSI2) -> MOSI
P1.4 (MISO2) -> MISO
P2.1 (PWM2) -> XDCS
P2.4 (PWM5) -> RST
P2.6 (CAP0) -> DREQ
P2.7 (CRD2) -> CS
VCC 3.3V -> VCC
GND -> GND

SJtwo to Volume Dial:
P0.15 (SCK0) -> CLK
P0.18 (MOSI0) -> DT
P0.11 (SCL1) -> SW
VCC 3.3V -> +
GND -> GND

SJtwo to Menu Dial:
P1.31 (ADC5) -> CLK
P1.20 (QEI.A) -> DT
P1.28 (CAP0) -> SW
VCC 3.3V -> +
GND -> GND

SJtwo to LCD Screen:
P2.8 (CTD2) -> RX1
VCC 3.3V -> RAW
GND -> GND
